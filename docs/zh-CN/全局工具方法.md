# 全局工具方法

本文将提供 Fragy 可用的全局工具方法的参考。

## parseArticle

```ts
parseArticle: (article: string) => ParsedArticle;

interface ArticleMeta {
  title: string; // 文章标题
  date: string; // 文章创建时间
  tags: string[]; // 文章标签
  categories: string | string[]; // 文章类别
}

interface ParsedArticle {
  meta: ArticleMeta;
  abstract: stirng; // 文章摘要
  content: string; // 文章内容
}
```
